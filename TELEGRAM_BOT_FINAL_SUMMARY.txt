1️⃣ УЗНАЙТЕ СВОЙ TELEGRAM ID:
   
   Вариант А (через бота):
   $ cd /Users/rex/Documents/cards/backend
   $ source venv/bin/activate
   $ python get_telegram_id.py
   
   → Откройте бота в Telegram
   → Напишите /start
   → Скопируйте ID
   → Ctrl+C для остановки
   
   Вариант Б (через @userinfobot):
   → Откройте https://t.me/userinfobot
   → Отправьте любое сообщение
   → Скопируйте ID

───────────────────────────────────────────────────────────────────────

2️⃣ ДОБАВЬТЕ СЕБЯ КАК АДМИНА:
   
   $ nano backend/telegram_bot/bot_admin.py
   
   Найдите строку 24 и замените:
   
   ADMIN_IDS = [
       123456789,  # <-- ВАШ ID СЮДА
   ]
   
   Сохраните: Ctrl+O → Enter → Ctrl+X

───────────────────────────────────────────────────────────────────────

3️⃣ ОСТАНОВИТЕ СТАРЫЙ БОТ:
   
   Найдите окно терминала где он запущен
   Нажмите: Ctrl + C

───────────────────────────────────────────────────────────────────────

4️⃣ ЗАПУСТИТЕ НОВЫЙ БОТ:
   
   $ cd /Users/rex/Documents/cards/backend
   $ source venv/bin/activate
   $ python manage.py run_new_bot
   
   Должно появиться:
   🤖 Бот запущен и готов к работе...

───────────────────────────────────────────────────────────────────────

5️⃣ ПРОВЕРЬТЕ РАБОТУ:
   
   → Откройте бота в Telegram
   → Напишите: /admin
   → Вы должны увидеть:
   
   🔐 Админ-панель
   [➕ Добавить новую карту]
   [📋 Мои карты]
   [📊 Статистика]
   
   Если видите - ВСЁ РАБОТАЕТ! ✅

───────────────────────────────────────────────────────────────────────

6️⃣ ДОБАВЬТЕ ПЕРВУЮ КАРТУ:
   
   → Нажмите "➕ Добавить новую карту"
   → Отправьте фото карты
   → Введите название
   → Введите описание (или /skip)
   → Получите QR-код! 📱
   → Отправьте фото упаковки (или /skip)
   → ГОТОВО! ✅

Q: Где взять Telegram ID?
A: https://t.me/userinfobot или python get_telegram_id.py

Q: Бот не отвечает на /admin?
A: Проверьте что добавили правильный ID в ADMIN_IDS

Q: Как добавить других админов?
A: Добавьте их ID в ADMIN_IDS через запятую

Q: Можно ли редактировать карты?
A: Да, через Django admin: http://localhost:8000/admin

Q: Сколько карт можно добавить?
A: Неограниченно!

Q: Что если потерял QR-код?
A: Скачайте заново через Django admin

Q: Клиент говорит "карта не найдена"?
A: Проверьте что карта активна в админке
